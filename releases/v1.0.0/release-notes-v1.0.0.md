# Greenpler v1.0.0 — Release notes

Release date: 2025-11-03

## Metadata
- Platform: Zepp OS
- Minimum API: 3.6
- Target: 480 (round)

## Changelog
- Initial release: lightweight watchface for Balance and Balance 2.
- Features: time (hrs/min), seconds, date & weekday with calendar shortcut, battery indicator with shortcut, hidden shortcuts (HR, steps, sleep), color picker (lazy-create).
- Performance: optimized updates and widget lifecycle to reduce memory and battery usage.
